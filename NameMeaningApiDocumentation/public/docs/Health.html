Health Check

Check API and database health status.

GET /api/v1/health

This endpoint has no parameters. Use it to check if the API is running.

Example Request

# Check API health
curl "https://namverse-api.vercel.app/api/v1/health"

# Legacy health endpoint
curl "https://namverse-api.vercel.app/health"

Example Response

{
  "status": "ok",
  "database": "connected",
  "timestamp": "2024-12-26T10:30:00.000Z",
  "version": "1.0.0"
}

Warmup Endpoint

Use this endpoint to warm up serverless functions (helps with cold starts):

GET /warmup 
# Warmup response
{
  "status": "warm",
  "totalMs": 150,
  "timestamp": "2024-12-26T10:30:00.000Z"
}
