Articles API

Live Testing
GET /api/v1/articles

Comprehensive articles API with pagination, filtering, sorting, full-text search, category management, and individual article retrieval by slug.

Description

The Articles API provides access to a rich collection of articles about names, cultures, naming trends, and traditions. Features include advanced pagination, category-based filtering, publication status control, multiple sorting options, powerful full-text search, category statistics, and SEO-friendly slug-based retrieval. Perfect for building content-rich naming applications and blogs.

Available Endpoints

GET /api/v1/articles

Get paginated list of articles with optional filters

GET /api/v1/articles/latest

Get latest published articles

GET /api/v1/articles/categories

Get all article categories with counts

GET /api/v1/articles/search

Search articles by query string

GET /api/v1/articles/:slug

Get single article by slug

Endpoint 13: GET /api/v1/articles

Description

Get paginated list of articles with optional filters for category, status, and sorting options.

Query Parameters

page optional

Page number (default: 1)

Type: integer | Default: 1
limit optional

Items per page (default: 10, max: 50)

Type: integer | Default: 10 | Max: 50
category optional

Filter by category (e.g., "Baby Naming Trends", "Cultural Traditions")

Type: string | Example: Baby Naming Trends
status optional

Filter by publication status (default: published)

Type: enum | Values: published, draft | Default: published
sort optional

Sort order for results (default: recent)

Type: enum | Values: recent, oldest, views, likes | Default: recent

Request Examples

Get first page with defaults:

GET /api/v1/articles

Get page 1 with 5 items:

GET /api/v1/articles?page=1&limit=5

Filter by category with limit:

GET /api/v1/articles?category=Baby%20Naming%20Trends&limit=10

Sort by views with limit:

GET /api/v1/articles?sort=views&limit=5

cURL Examples

curl "http://localhost:5000/api/v1/articles"
curl "http://localhost:5000/api/v1/articles?page=1&limit=5"
curl "http://localhost:5000/api/v1/articles?category=Baby%20Naming%20Trends&limit=10"
curl "http://localhost:5000/api/v1/articles?sort=views&limit=5"

Endpoint 16: GET /api/v1/articles/search

Description

Search articles by query string with full-text search capability.

Query Parameters

q required

Search query (required, min: 2 characters)

Type: string | Min Length: 2 characters
page optional

Page number (default: 1)

Type: integer | Default: 1
limit optional

Items per page (default: 10, max: 50)

Type: integer | Default: 10 | Max: 50

Request Examples

Search for "name" with limit 5:

GET /api/v1/articles/search?q=name&limit=5

Search for "islamic" with pagination:

GET /api/v1/articles/search?q=islamic&page=1&limit=10

Search for "meaning":

GET /api/v1/articles/search?q=meaning

Endpoint 17: GET /api/v1/articles/:slug

Description

Get single article by its unique slug identifier.

Path Parameters

slug required

Article slug (lowercase, hyphenated)

Type: string | Format: lowercase-with-hyphens | Example: modern-noahkwvrin

Request Examples

Get article by slug:

GET /api/v1/articles/modern-noahkwvrin

Get another article:

GET /api/v1/articles/cultural-noramp7l47
