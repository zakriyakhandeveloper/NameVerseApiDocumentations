Articles Categories API

Live Testing
GET /api/v1/articles/categories

Get all article categories with article counts for each category. Perfect for building navigation menus and category filters.

Description

The Articles Categories API returns a complete list of all article categories available in the NameVerse database, along with the count of articles in each category. This endpoint requires no parameters and provides instant results, making it ideal for building category navigation menus, filter dropdowns, and analytics dashboards.

Query Parameters

No query parameters required for this endpoint.

Simply call the endpoint to receive all categories with their counts.

Request Examples

Get all categories:

GET /api/v1/articles/categories

cURL Example

curl "https://namverse-api.vercel.app/api/v1/articles/categories"

Response Fields

Field Type Description
success boolean Indicates if the request was successful
count integer Total number of categories
data array Array of category objects
data[]._id string Category name
data[].count integer Number of articles in this category

JavaScript Example

// Fetch categories and create a navigation menu
async function buildCategoryMenu() {
  try {
    const response = await fetch(
      'https://namverse-api.vercel.app/api/v1/articles/categories'
    );
    const data = await response.json();

    if (data.success) {
      const menuHTML = data.data.map(category => `
        
          ${category._id} (${category.count})
        
      `).join('');

      document.getElementById('categoryMenu').innerHTML = menuHTML;
    }
  } catch (error) {
    console.error('Error fetching categories:', error);
  }
}

buildCategoryMenu();

Integration with Other Endpoints

Use category names from this endpoint to filter articles:

Get articles in a specific category:

GET /api/v1/articles?category=Baby%20Naming%20Trends

Search within a category:

GET /api/v1/articles/search?q=popular&category=Modern%20Names
