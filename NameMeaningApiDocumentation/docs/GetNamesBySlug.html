Get Name by Religion & Slug

Live Testing
GET /names/{religion}/{slug}

Get detailed name information using SEO-friendly URL structure with religion and slug identifier.

Description

This endpoint retrieves detailed information for a specific name using SEO-friendly URLs. Navigate to /names/{religion}/{slug} to view name details dynamically with optimized SEO meta tags.

Path Parameters

religion string required

The religion category (lowercase).

Available values
islamic christian hindu
slug string required

The unique slug of the name (lowercase).

Examples
kamran muhammad aisha

URL Examples

/names/islamic/kamran
/names/islamic/muhammad
/names/christian/john

Response

200 Success 
{
  "name": "Kamran",
  "slug": "kamran",
  "meaning": "Successful, Prosperous",
  "origin": "Persian",
  "gender": "male",
  "religion": "islamic",
  "lucky_number": 5,
  "lucky_day": "Friday"
}
404 Not Found 
{
  "error": "Name not found"
}

Features

SEO-optimized URLs with dynamic meta tags
Auto-fetches and displays name data on page load
Beautiful card display with gradient design
Shareable URLs for each name
Includes additional info like lucky day, personality traits
