Names (v1)

Live Testing
GET /api/v1/names

Fetch names with pagination and filtering across religions and origins.

Query Parameters

religion required

islamic, christian, hindu

page optional

Default: 1

limit optional

Default: 50, Max: 100

gender optional

male, female

origin optional

Filter by origin (e.g., Arabic, Hebrew)

startsWith optional

Filter by starting letter (A-Z)

search optional

Search in name and meaning

sort optional

asc, desc

Examples

/api/v1/names?religion=islamic /api/v1/names?religion=islamic&gender=male&limit=10 /api/v1/names?religion=islamic&search=ali /api/v1/names?religion=islamic&startsWith=A&limit=20 /api/v1/names?religion=islamic&page=2&limit=50
