GET /api/v1/search

Search for names using a query string with partial matching and flexible filtering.

Description

Search for names using a query string. Supports partial matching, case-insensitive search, and returns matching names across all religions. Includes optional filtering by religion and limit control.

Query Parameters

q string required

Search query string (minimum 1 character). Names that start with this query will be returned.

Examples
sa muh john
limit integer optional

Maximum number of search results to return.

Default
20
Minimum
1
Maximum
100
religion string optional

Filter results by religion category.

Available values
islamic christian hindu

Request Examples

GET /api/v1/search?q=sa&limit=5
GET /api/v1/search?q=muh&limit=10&religion=islamic
GET /api/v1/search?q=john
