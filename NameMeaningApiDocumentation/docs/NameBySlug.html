Get Name by Slug

Get detailed information about a specific name by its slug and religion.

GET /api/v1/names/:religion/:slug

Path Parameters

ParameterTypeRequiredDescription
religionstringYesislamic | christian | hindu
slugstringYesName slug (lowercase, hyphenated)

Example Requests

# Get Islamic name "Ali"
curl "https://namverse-api.vercel.app/api/v1/names/islamic/ali"

# Get Christian name "Mary"
curl "https://namverse-api.vercel.app/api/v1/names/christian/mary"

# Get Hindu name "Arjun"
curl "https://namverse-api.vercel.app/api/v1/names/hindu/arjun"
