Articles API

Live Testing
GET /api/v1/articles

Get paginated articles with filtering, search, categories, and individual article details.

Description

The Articles API provides access to a comprehensive collection of articles with support for pagination, filtering by category and status, sorting options, full-text search, and individual article retrieval.

Available Endpoints

GET /api/v1/articles

List articles with pagination and filters

GET /api/v1/articles/latest

Get most recent articles

GET /api/v1/articles/categories

Get all article categories

GET /api/v1/articles/search

Search articles by keyword

GET /api/v1/articles/:slug

Get single article by slug

1. List Articles

Query Parameters

page optional

Page number for pagination (default: 1)

limit optional

Items per page (default: 10, max: 50)

category optional

Filter by category

status optional

published or draft (default: published)

sort optional

recent, oldest, views, or likes (default: recent)

Examples

/api/v1/articles /api/v1/articles?page=2&limit=20 /api/v1/articles?category=islamic&sort=views

4. Search Articles

Query Parameters

q required

Search query string

page optional

Page number (default: 1)

limit optional

Items per page (default: 10, max: 50)

Example

/api/v1/articles/search?q=islamic+names&limit=5

Response

{
  "success": true,
  "query": "islamic names",
  "data": [
    {
      "id": 1,
      "title": "Popular Islamic Names",
      "slug": "popular-islamic-names",
      "excerpt": "Discover the most beloved...",
      "category": "islamic",
      "relevance": 0.95
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 5,
    "total": 12
  }
}
