Introduction

The NameVerse API provides access to a comprehensive database of 70,000+ names across Islamic, Hindu, and Christian traditions. Each name includes meanings, origins, gender, and cultural information.

Base URL

https://namverse-api.vercel.app/api/v1

70,000+ Names

Comprehensive database

~100ms Response

Lightning fast

3 Religions

Islamic, Hindu, Christian

Authentication

No Authentication Required

The NameVerse API is completely free and open. No API keys, tokens, or registration required. Simply make HTTP requests to the endpoints.

Rate Limits

Limit Type Value Description
Daily Limit 100 requests Per IP address, resets at midnight UTC
Burst Limit 10 requests/sec Maximum concurrent requests
Response Time ~100ms avg Typical API response time

When rate limit is exceeded, API returns 429 Too Many Requests. Wait until the limit resets or reduce request frequency.

Error Handling

The API uses standard HTTP status codes. All error responses include a JSON body with details.

Status Code Meaning Description
200 OK Request successful
400 Bad Request Invalid parameters or missing required fields
404 Not Found Resource not found (name, article, etc.)
429 Too Many Requests Rate limit exceeded
500 Server Error Internal server error

Error Response Format

{
  "success": false,
  "error": "Name not found",
  "message": "The requested name 'xyz' does not exist in our database",
  "statusCode": 404
}

Get Names

Live Test

Retrieve a paginated list of names with optional filters. Supports filtering by religion, gender, origin, and starting letter.

GET /api/v1/names

Query Parameters

Parameter Type Default Description
religion string islamic Filter by religion: islamic, christian, hindu
page integer 1 Page number for pagination
limit integer 50 Results per page (max: 100)
gender string - Filter by gender: male, female
origin string - Filter by origin (e.g., Arabic, Hebrew)
startsWith string - Filter names starting with letter
sort string asc Sort order: asc, desc

Example Request

# Get Islamic male names
curl -X GET \
  "https://namverse-api.vercel.app/api/v1/names" \
  -d "religion=islamic" \
  -d "gender=male" \
  -d "limit=10"

Response Structure

{
  "success": true,
  "religion": "islamic",
  "data": [
    {
      "_id": "...",
      "name": "Muhammad",
      "slug": "muhammad",
      "gender": "Male",
      "origin": "Arabic",
      "short_meaning": "Praised"
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 10,
    "total": 22720,
    "totalPages": 2272,
    "hasMore": true
  }
}
GET

Search Names

Live Test

Full-text search across all names. Search by name or meaning. Minimum 2 characters required.

GET /api/v1/names/search

Query Parameters

Parameter Type Required Description
q string Yes Search query (min: 2 characters)
religion string No Filter by religion
limit integer No Results limit (default: 20, max: 50)

Example

curl "https://namverse-api.vercel.app/api/v1/names/search?q=ali&religion=islamic&limit=10"

Get Name by Slug

Live Test

Get complete details for a specific name including meaning, origin, lucky number, personality traits, and famous people with this name.

GET /api/v1/names/:religion/:slug

Path Parameters

Parameter Type Description
religion string Religion: islamic, christian, hindu
slug string Name slug (lowercase, e.g., "muhammad", "john")

Response Fields

name - Full name
meaning - Name meaning
gender - Male/Female
origin - Name origin
lucky_number - Lucky number
lucky_day - Lucky day
personality - Personality traits
famous_people - Famous people

Example

curl "https://namverse-api.vercel.app/api/v1/names/islamic/muhammad"
